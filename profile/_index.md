## 完整度说明

- **高**：事实充分、有现成表述，可直接进简历
- **中**：主干事实有，但面试官视角还有明显空洞，建议深挖补全
- **低**：只有一两句话，用之前必须深挖

## 待办（全局待确认事项）

- [ ] 首次使用（有简历）：把简历发给 agent 说「帮我建库」，它会走沉淀工作流把这里填满
- [ ] 首次使用（没有简历）：直接说「我没有简历，帮我从零开始」，agent 会访谈式普查素材建库，并生成你的第一份标准版简历
