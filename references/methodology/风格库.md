# 风格库：公司性质决定证据排序

> 写作方法是骨架，公司性质只决定先展示什么证据、使用什么语气。先执行《简历方法论》的保真与价值翻译，再按目标公司选择本文件对应章节；发生冲突时，以保真和方法论为准。

本文件只管企业层的整体调性，不代替岗位规则。使用顺序固定为：通用方法 → 本文件对应公司章节 → 《岗位速查》的对应岗位或专项文件。比如“大厂行研”不是只套“大厂风”，还必须读取行研专项；“外企财务”则叠加外企与财务两套规则。

## 券商 / 银行 / 金融机构

**一句话**：专业、严谨、数字精确。HR 用证书和课程验真，用数字口径判断你严不严谨。

- **数字是信任货币**：单位（万/亿/bps/pp）、时间口径（同比/环比）、统计口径（净额/总额）三全。拿不准的数字宁可不写——「约 1 亿」比不写更减分。
- **证书分级写**：写「通过 FRM 一级」而不是「FRM 证书」，写「CFA 一级（备考二级）」而不是「备考 CFA」。从业资格类合并一行。证书栏位置靠前。
- **课程是背书**：列 3-5 门金融核心课（公司金融、投资学、计量经济学、财务报表分析方向），有高分可附分数。
- **项目模型化**：背景 → 个人负责的模块 → 方法与数据源（估值方法名、Wind/iFinD）→ 产出与是否被采纳。术语必须准确——市盈率/市净率级别的混用等于直接出局。
- **实习写专业动作**：研究、风控、投行类实习突出独立交付物；操作类实习突出业务量与零差错。
- **分支**：中资大行的招聘逻辑向国央企靠（见下节）；外资机构叠加外企规则（英文版、STAR），且对 CFA/FRM 权重更高。
- **忌**：「对金融充满热情」类空话、创新颠覆叙事、无量化的经历。

## 大厂（互联网 / 科技）

**一句话**：HR 几秒扫一份简历，数字是锚点，JD 的语言是通行证。

- **每段经历至少 2 个数字**：一个业务规模（用户数/单量/篇数），一个个人产出（提升幅度/覆盖范围）。「200+ 篇」远胜「大量」。
- **动词有力但保真**：真主导才写「主导」；协助的写「支持 / 配合推进」，不硬拔头衔和职责。
- **JD 关键词自然植入**：用 JD 反复出现的词重写你真做过的事，一条 bullet 命中 1-2 个即可，堆砌会露馅。
- **经历项目化**：时间、背景、角色、动作、量化五要素齐；社团活动也这么包装。
- **归属写法**：在大集团子业务实习写「母公司 / 业务线」；独立小公司不挂靠。
- **公司口味微调**：字节看数据与实验思维；阿里看商业逻辑与价值观契合；腾讯看产品体验感；美团/拼多多看实操落地；内容平台（小红书/B 站）看你是不是它的深度用户。
- **忌**：「责任心强 / 沟通能力强 / 获领导认可」类空话、工具图标堆砌、拿浅度大厂经历硬撑门面。

## 国央企

**一句话**：完整、稳重、能融入体系——与大厂逻辑几乎相反，同一份简历不能混投。

- **信息求全**：政治面貌必写（是群众也写，不写显得回避）；籍贯、出生年月可写；教育信息全列不删减。
- **学生干部、志愿服务、社会实践是核心资产**：前置展开写（职务、任期、协调规模、活动清单）——这些在大厂简历里要压缩的，在这里要放大。
- **荣誉选主旋律**：奖学金、三好/优秀学生干部、挑战杯/互联网+ 权重高于商业比赛。
- **语气谦逊**：多用「参与 / 协助 / 在指导下完成」，少个人英雄主义；数字要有但表述克制。
- **技能保守写**：Office 可以拆开写（这里不嫌啰嗦嫌花哨），少列潮流工具与黑话。
- **政审意识**：全文档零虚假、海外经历如实——简历漏洞会在政审环节被放大。
- **忌**：颠覆/创新叙事、突出个人商业成就、体现出「不稳定」的痕迹。

## 外企

**一句话**：STAR 结构 + 强动词 + 国际可读的量化；应届英文简历严格一页。

- **每条 bullet 过去式强动词开头**（Led / Built / Analyzed / Reduced / Delivered），禁用 Helped / Assisted / Responsible for；全文不出现 "I"。
- **STAR 完整**：背景、目标、动作、结果四要素，结果必须量化。
- **量化国际化**：M/K 计数、YoY/QoQ、bps；币种按公司全球性选择。
- **Skills 分栏**：Hard Skills / Tools（标熟练度）/ Languages(标水平与分数) / Certifications。
- **国际信号前置**：交换、全英文授课项目、外语等级、国际比赛。
- **双语策略**：英文版不是中文版的直译——英文更直接、更突出 result；中文版按目标企业性质另走对应规则。
- **Cover Letter**：咨询和外资金融机构权重高，「为什么是这家公司」必须具体到业务。
- **忌**：Chinglish、时态混用、超页。
